# AI Healthcare Assistant

Minimal Next.js 14 starter app with Tailwind CSS and a simple healthcare-themed UI.

Quick start

1. Install dependencies

```bash
cd healthcare-assistant
npm install
```

2. Run the development server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

Notes

- This project uses Tailwind CSS. If the editor shows CSS 'Unknown at rule @tailwind' warnings, that's expected until PostCSS/Tailwind are installed in your environment.
- To build for production: `npm run build` and `npm start`.
